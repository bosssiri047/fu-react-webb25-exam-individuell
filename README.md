# fu-react-webb25-exam-individuell
## Which external libraries did I choose?
I picked Swiper, Motion, Fontawesome, UUID, and Barcode.
## Why did I pick these?
I believe these five libraries are perfectly suited to what the app aims to achieve.
Swiper and Motion are used to help with presentation and make elements of the app more interactive.
Fontawesome is used for importing icons such as the cart icon and the back arrow icon, so that the user experience is smoother.
UUID and Barcode are used for assigning a unique for each ticket receipt.
## How did I use these libraries?
For Swiper, I used it on the receipt page to render the ticket receipt in a card format, where the user can swipe right to see the next ticket or left to see the previous one.
For Motion, I used it on the buttons so they animate when the user presses them, giving the user feedback for their action.
For Fontawesome, I used icons from the library to help with design language on the user interface to improve the user experience.
For UUID, I used it to assign each receipt of tickets its own unique identification number.
For Barcode, it's used in tandem with UUID so that it renders out a unique barcode for each assigned UUID on the receipt.
